# Astro Claude Skills v0.1.0

Initial release of a Claude Code skill suite for premium Astro website work.

Included:

- reference audit workflow
- capsule composition workflow
- Astro build workflow
- final polish workflow
- generic capsule catalog and page recipes

This release is intentionally generic:

- no client names
- no source site names
- no brand identities from imported references

The goal is to help Claude build high-quality Astro sites from structural patterns, not to clone source websites.
