# Audit workflow

1. Inspect the important pages, navigation, anchors, assets and visible hierarchy.
2. List the public routes and section order of each important page.
3. Separate:
   - must-keep text
   - must-keep images
   - structural behaviors worth preserving
   - legacy artifacts to discard
4. Map each section to:
   - an existing capsule
   - a site-only component
   - a future reusable capsule
5. Note delivery constraints:
   - root-route or nested-route target
   - preview gating requirement
   - metadata / AI SEO requirement
   - preview or handoff packaging requirement
