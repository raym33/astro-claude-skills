# Output format

## Source
- path or URL
- what was inspected

## IA to preserve
- pages
- anchors
- navigation logic

## Must-keep content
- headings
- body copy
- image groups

## Capsule mapping
- section -> capsule

## Bespoke components
- section -> site-only component

## Discard
- widgets, builder leftovers, broken interactions, etc.

## Build recommendation
- route strategy
- asset strategy
- implementation order
