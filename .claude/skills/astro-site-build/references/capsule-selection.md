# Capsule selection

Use the lightest section that preserves the source structure:

- opening statement -> `hero`
- service overview -> `service-map` or `resource-cards`
- alternating story rows -> `feature-pair`
- process or method -> `sticky-steps`
- segmented explanation -> `product-tabs`
- proof interruption -> `proof-strip`
- long-form human proof -> `testimonial-stories`
- authored quote -> `editorial-quote`
- FAQ / answer section -> `faq-list`
- closing conversion -> `join-cta`

If none of these preserve the section honestly, build a site-only component.
